About Me

Hi I'm Sebastian Morton-Zeller I built this page as an interactive way to help bolster my resume while strengthening my skills in popular web development languages and testing my knowledge of Cloud development.

The full stack behind this page is as follows:

  • HTML/CSS (obviously) to make the page look decent, along with some JavaScript/jQuery to control animations
  • Amazon S3 - deployed as a static page in an S3 bucket
  • Amazon CloudFront - provides content delivery/web caching as well as HTTPS
  • Amazon Route 53 + NameCheap.com - DNS routing to the CloudFront distrubution
  • More JavaScript - created a custom visitor counter to keep track of how many people have accessed the site
  • Amazon DynamoDB - used to store, retrieve, and update the count for the visitor counter
  • Amazon Boto3 Library(Python)/Lambda - used to programmatically manage and automate serverless functions + create robust testing
  • Amazon Serverless Application Model(SAM) tempalte/Terraform - Infrastructure as Code (IaC)
  • GitHub Actions - provides a CI/CD pipeline/workflow to automate the testing and deployment of updates to the site as new code gets pushed to the repository
